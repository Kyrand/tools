# tools.kyrandale.com

Assorted useful browser-based tools.

Live at **[tools.kyrandale.com](https://tools.kyrandale.com)**.

Each tool is a single self-contained HTML file. The build pipeline (inspired by [simonw/tools](https://github.com/simonw/tools)) generates an index, colophon, and per-file footer from git history.
